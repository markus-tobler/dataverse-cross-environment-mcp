# Support

## How to file issues and get help

This project uses GitHub Issues to track bugs and feature requests. Please search the existing issues before filing new issues to avoid duplicates.

For new issues, file your bug or feature request as a new Issue in the GitHub repository.

## Microsoft Support Policy

Support for this project is limited to the resources listed above. This is a community-driven project and is not covered by Microsoft Support.

For questions and discussions, please use GitHub Discussions or GitHub Issues.
